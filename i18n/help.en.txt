*woltsapp* - WhatsApp Wake-on-LAN (WoL) bot

Commands:
  - *`help`*: print this help page
  - *`addGroup`*: allowlist the current group so commands can be run here
  - *`removeGroup`*: remove the group from the allowlist
  - *`wake|wol`* `<hostname or MAC>`: send a WoL magic packet to the specified host
  - *`addHost`* `<name>` `<MAC>`: add an alias for a MAC address
  - *`removeHost`* `<name>`: remove an alias
  - *`hosts`*: list all defined aliases
